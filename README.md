# JavaPrintFieldAndClassFromJson

* 介绍：根据给出的json来打印（生成）出自命名的类<br>
* 可以根据json_to_field方法打印（生成）java属性<br>
* 可以根据json_to_class方法打印（生成）包含解析的java类，但目前只支持int和String类型<br>
